# ExpressJs-MongoDB-Todo-API

A simple Todo API built with Express.js and MongoDB, featuring user authentication with JWT and bcrypt.

## Features

- User authentication with JWT
- Create, read, update, and delete todo items

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (jsonwebtoken)
- bcrypt

### Prerequisites
